## about datathon
**<평가 방식>** 

- EDA 및 시각화가 적절하게 이루어졌는가? (15%)
- 데이터 분석을 통한 인사이트가 도출이 잘 되었는가? (15%)
- 도출한 인사이트를 통해 설득력이 충분하였는가? (20%)
- 팀원들이 협업을 통해 프로젝트를 수행하고 발표를 진행하였는가? (25%)
- 발표시간을 준수하였는지? (25%)
- 가점 : 교육생들의 따봉 (좋아요 수치를 합산하여 가산점으로 반영)
